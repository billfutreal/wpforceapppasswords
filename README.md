# wpforceapppasswords
For Hostinger hosting - force the ability to allow creation of application passwords

Hostinger blocks the setting of appication passwords from the user admin profile - this plugin overrides that block with no side effects.  
## Usage
Upload to wp-plugins
Go to WP plugin in your dashboard and activate
Log out and log back in to Wordpress
Go to your user profile
You'll now see the option to create and appication password
